# Customer Behaviour Data Analyst Project

# 📌 Project Overview
This project focuses on analyzing customer shopping behavior using Python and SQL to uncover meaningful business insights. The analysis helps identify purchasing patterns, customer preferences, spending habits, and factors influencing buying decisions. The goal is to support data-driven business decisions through data analysis and interactive visualizations.

# 🎯 Objectives
- Analyze customer purchasing behavior.
- Identify high-value customers and shopping trends.
- Perform data cleaning and exploratory data analysis (EDA).
- Generate business insights using SQL queries.
- Create an interactive dashboard for better visualization.

# 🛠️ Technologies Used
- Python
- SQL
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI / Excel Dashboard

# 📂 Project Files
- Customer Shopping Behavior Dataset
- SQL Queries
- Business Problem Document
- Dashboard
- README

# 📊 Key Insights
- Customer purchase patterns by age and gender.
- Most popular product categories.
- Spending behavior across different customer segments.
- Revenue trends and customer preferences.
- Actionable recommendations for business growth.

# 🚀 Conclusion
This project demonstrates how data analytics can transform raw customer data into valuable business insights. The findings can help organizations improve customer satisfaction, optimize marketing strategies, and increase overall sales performance.
